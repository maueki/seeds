# SEEDS

## About

SEEDS(Simple and Easy Event Driven State machine) is a C++ header only state machine library based on [libev](http://software.schmorp.de/pkg/libev.html).

## Requirements

* [libev](http://software.schmorp.de/pkg/libev.html)
* C++11 or higher
